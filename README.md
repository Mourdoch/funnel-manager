# Funnel Manager

Static GitHub Pages version of Funnel Manager.

## Publish with GitHub Pages

1. Create a public GitHub repository named `funnel-manager`.
2. Upload all files in this folder to the repository root.
3. Go to `Settings > Pages`.
4. Set source to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

The share link will look like:

```text
https://YOUR-GITHUB-USER.github.io/funnel-manager/
```

## Security check

This copy is static HTML, CSS, JavaScript, and image assets only. No secrets, API keys, access tokens, passwords, backend URLs, or database credentials were found during the pre-publish scan.
